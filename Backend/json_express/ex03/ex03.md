3) É possível considerar a solução do item 2 “segura”, já que transmite a
senha via JSON, de forma privada? Explique

Não. Transmitir a senha em JSON não torna a comunicação segura por si só.

O JSON é apenas um formato de organização dos dados. Se a requisição for enviada por HTTP comum, qualquer pessoa que intercepte o tráfego poderá ler a senha em texto puro.
